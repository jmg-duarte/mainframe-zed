# Mainframe

A light theme for [Zed](https://zed.dev) that brings back the mainframe vibes to your editor.

Mainframe uses a warm, paper-like background with a restrained color palette:

- **Brown** (`#884400`) for keywords and control flow
- **Green** (`#226600`) for strings, numbers, and constants
- **Blue** (`#0054a6`) for types, traits, and interfaces
- **Purple** (`#660099`) for macros, attributes, and preprocessor directives
- **Black** (`#000000`) for variables, functions, and operators

## Installation

Search for "Mainframe" in the Zed extension marketplace, or install it from the command palette with `zed: install extension`.
